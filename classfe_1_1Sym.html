Detailed Description

A Symbol just wraps a const std::string*, so pass Sym itself around as value.

With the exception of the empty string, you should only create Symbols via SymPool::sym. This in turn will toss all Symbols into a big hash set. This makes Sym::operator== and Sym::operator!= an O(1) operation. The empty string is internally handled as nullptr. Thus, you can create a Symbol representing an empty string without having access to the SymPool.

Note
The empty std::string, nullptr, and "\0" are all identified as Sym::Sym().
